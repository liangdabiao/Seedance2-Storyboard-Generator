## 片段1：希望升华（0-8s）

### 方案A：视频延长（优先）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @视频1 | E04视频 | 延长源 |
| @图片1（参考） | **C06** | 小晚·晨光仰拍 -- 造型参考 |

#### Seedance Prompt

```
将@视频1延长，电影感治愈风格，9:16竖屏
@图片1角色参考

【镜头】窗边 → 走出大楼 → 仰拍镜头上摇至天空
【主体】小晚走向户外，仰望晨光
【变化】室内晨光 → 步出大门 → 阳光破云而出
【光影】金色阳光渐强，希望之光
【声音】温暖全奏渐强 + 鸟鸣 + "温柔对待的瞬间都会留下回响"
```

### 方案B：独立生成（备用）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @图片1（首帧） | **KF09** | 走出大楼 -- 仰拍晨光 |
| @图片2（参考） | **C06** | 小晚·晨光仰拍 -- 造型参考 |

#### Seedance Prompt

```
电影感治愈风格，9:16竖屏，宏大温暖
@图片1首帧，@图片2角色参考

【镜头】低角度仰拍 → 人物望向天空 → 上摇至天空
【主体】小晚走出大楼，仰望晨光
【变化】大楼阴影中 → 步入阳光 → 天空破云
【光影】金色阳光破云，希望之光
【声音】温暖全奏渐强 + 城市晨声 + "晚安，早安"
```

### 尾帧描述
纯净天空，阳光破云而出，光芒万丈。没有人物，只有温暖的阳光和云朵，象征治愈与希望的升华。

### 导演备注
- **情绪基调**: 治愈、希望、升华
- **关键动作**: 从人物仰望到镜头上升至天空，完成从个人到普世情感的升华
- **色调提示**: 明亮温暖金色调，阳光充足，象征阴霾散去
- **结尾设计**: 尾帧纯天空画面，留给观众想象空间

## 全剧总结

### 情感曲线
```
E01(静谧孤独) -> E02(忧伤回忆) -> E03(触动转折) -> E04(温暖治愈) -> E05(希望升华)
    暖调微光      冷蓝回忆      冷暖交融       全暖治愈        光明结尾
```

### 推荐生成顺序
```
E01(独立首帧) -> E02(延长E01) -> E03(延长E02) -> E04(延长E03) -> E05(延长E04)
```
如果某集视频延长效果不理想，切换到方案B独立首帧生成，后续集继续用该集视频延长。

### 首尾呼应
- **开场**: 深夜11点，小晚独自开播
- **结尾**: 清晨阳光，小晚走向希望
- **主题**: 从深夜到黎明，从孤独到被看见，从伤痛到治愈
