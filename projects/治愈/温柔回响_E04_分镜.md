# 温柔回响 E04 分镜脚本
## 回信与和解（8秒）

### 制作流程概览

| 片段 | 时长 | 生成方式 | 上传素材 | 镜头概要 |
|------|------|----------|----------|----------|
| 片段1 | 8s | 方案A：视频延长（优先） / 方案B：首帧生成 | E03视频 或 KF07首帧 | 回信书写与释然 |

---

## 片段1：回信释然（0-8s）

### 方案A：视频延长（优先）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @视频1 | E03视频 | 延长源 |
| @图片1（参考） | **C05** | 小晚·释然微笑 -- 表情参考 |

#### Seedance Prompt

```
将@视频1延长，电影感治愈风格，9:16竖屏
@图片1角色参考

【镜头】面部特写 → 拉远至手部书写 → 望向窗外
【主体】小晚写回信，笔触温柔
【变化】微笑低头 → 专注书写 → 抬头望阳光释然
【光影】金色晨光洒满桌面，温暖治愈
【声音】治愈吉他 + 书写声 + "谢谢你看见了我"
```

### 方案B：独立生成（备用）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @图片1（首帧） | **KF07** | 回信书写 -- 晨光中手写 |
| @图片2（参考） | **C05** | 小晚·释然微笑 -- 表情参考 |

#### Seedance Prompt

```
电影感治愈风格，9:16竖屏，金色晨光
@图片1首帧，@图片2角色参考

【镜头】手部特写 → 拉远现人 → 望向窗外阳光
【主体】小晚写回信，笔触温柔
【变化】专注书写 → 停顿微笑 → 望向阳光释然
【光影】金色晨光洒满桌面，温暖治愈
【声音】治愈吉他 + 书写声 + "被看见就是最好的治愈"
```

### 尾帧描述
小晚望向窗外初升的太阳，释然微笑。回信放在桌上，阳光透过信纸形成温暖光晕。

### 导演备注
- **情绪基调**: 温暖、释然、治愈
- **关键动作**: 从书写的专注到望向窗外的放松，体现内心和解
- **色调提示**: 全暖金色调，阳光充满画面，治愈感达峰值
- **衔接提示**: 望向窗外动作自然过渡到E05走出大楼
