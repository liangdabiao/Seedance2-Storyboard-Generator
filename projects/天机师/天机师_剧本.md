# 天机师 — 剧本

## 项目信息

| 项目 | 内容 |
|------|------|
| **题材** | 仙侠 / 悬疑 / 命运 |
| **风格** | 中国水墨画风格混合动画质感（水墨仙侠） |
| **画幅** | 9:16 竖屏 |
| **核心立意** | 明知窥探天机会折损自身，却仍愿在迷雾中为迷途者点亮一盏灯，然后在灯火燃尽前，悄然转身离去 |
| **整体调性** | 神秘、空灵、苍凉、宿命感 |

---

## 角色简介

### 天机师
- **外貌**：白发如雪的青年道人，面容清隽年轻，眼神却透着千年沧桑
- **服饰**：墨青色道袍，袍袖宽大随风而动，衣缘有淡金色天机纹路
- **标志物**：手持古铜天机盘（命盘），盘面刻有星宿符文，运转时发出幽蓝微光
- **气质**：超然物外，悲悯而克制，举止间带有"已知结局仍赴之"的从容
- **代价**：每窥探一次天机，白发便会黯淡一缕，生命烛火随之减损

## E01 — "雾中灯"

### 概要
**时长**：15秒
**情绪**：神秘 → 空灵 → 苍凉
**叙事功能**：角色出场，建立世界观与人物核心气质

### 剧情梗概

大雾弥漫的仙山之巅，万籁俱寂。远处一点幽蓝微光若隐若现——那是天机盘的光芒。

镜头推近，一位白发青年道人端坐于悬崖古松之下，面前的天机盘缓缓旋转，符文浮光流转。他缓缓抬眸，目光穿透迷雾，仿佛看见了某个人的命运。

天机盘骤然亮起，星宿符文腾空而起环绕在他周身。他的白发在光芒中微微黯淡——这是窥天机的代价。

他起身，收起天机盘，提起身旁一盏孤灯。转身走入浓雾深处，灯火渐远渐小，直至只剩一点微光。

### 起始画面
浓雾中的仙山剪影，水墨渲染，远处一点幽蓝光芒

### 结尾画面（尾帧）
天机师背影消失在浓雾中，只剩一盏孤灯的微弱光点，画面大面积留白水墨雾气
