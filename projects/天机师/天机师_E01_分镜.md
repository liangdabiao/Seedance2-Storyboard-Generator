# 天机师 E01 — "雾中灯"

## 制作流程概览

E01共15秒，拆分为**3个片段×5秒**。片段1独立生成，片段2-3优先视频延长。

| 片段 | 时间 | 生成方式 | @图片1 / @视频1 | @图片2 / @图片1(参考) | 镜头概要 |
|------|------|----------|-----------------|----------------------|----------|
| 片段1 | 0-5s | 首帧生成 | KF01 雾山幽光 | C01 天机师·正面全身 | 雾山推近 → 揭示道人端坐 → 天机盘发光 |
| 片段2 | 5-10s | 视频延长 | 片段1视频 | C02 天机师·面部特写 | 天机盘骤亮 → 符文腾空 → 白发黯淡 → 起身提灯 |
| 片段3 | 10-15s | 视频延长 | 片段2视频 | C03 天机师·背影 | 转身入雾 → 背影渐隐 → 只剩灯光远去 |

## 片段1：雾山揭示（0-5s）

### 上传素材
| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @图片1（首帧） | **KF01** | 首帧·雾山幽光 — 浓雾仙山远景，远处一点幽蓝微光 |
| @图片2（参考） | **C01** | 天机师·正面全身 — 角色形象参考 |

### Seedance Prompt
```
水墨仙侠风格混合动画质感，9:16竖屏
@图片1首帧，@图片2角色形象参考

【镜头】缓慢推镜头，从远景推至中景
【主体】浓雾仙山群峰，远处幽蓝微光，古松下白发道人端坐，天机盘悬浮
【变化】雾气流动 → 镜头穿雾推近 → 光源逐渐清晰 → 揭示道人与旋转天机盘 → 盘面符文开始发光
【光影】幽蓝微光从天机盘向外扩散，从下方照亮道人面部
【声音】空灵古琴 + 山风呼啸 + 天机盘转动声
```

### 尾帧描述
白发道人端坐古松下，天机盘悬浮旋转加速，幽蓝符文光芒逐渐增强，光点环绕道人周身

## 导演备注

1. **节奏**：整体偏缓，留白意境
2. **色调**：墨灰+幽蓝为主色，孤灯暖黄为唯一暖色——"冷中一点暖"
3. **光源叙事**：片段1幽蓝天机盘光 → 片段2冷暖交替 → 片段3暖黄孤灯独行，光源转换即角色弧线
4. **留白**：片段3结尾大面积水墨留白，"计白当黑"
5. **敏感词规避**：避免"修仙""渡劫""天道"等词
