## 片段1：雨夜等待（0-8s）

### 方案A：视频延长（优先）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @视频1 | E01视频 | 延长源 |
| @图片1（参考） | **C03** | 小晚·回忆雨天造型 -- 角色参考 |

#### Seedance Prompt

```
将@视频1延长，电影感治愈风格，9:16竖屏
@图片1角色参考

【镜头】窗外雨景 → 拉远至雨中街道全景
【主体】场景切换至回忆，女孩撑透明伞
【变化】窗上雨滴 → 街道雨夜 → 便利店前等待
【光影】冷蓝回忆色调，便利店霓虹映射
【声音】忧伤大提琴 + 雨声渐强 + "那年的雨下得很大"
```

### 方案B：独立生成（备用）

| 槽位 | 素材编号 | 内容 |
|------|----------|------|
| @图片1（首帧） | **KF03** | 雨中回忆 -- 透明伞下侧影，便利店霓虹 |
| @图片2（参考） | **C03** | 小晚·回忆雨天造型 -- 角色参考 |

#### Seedance Prompt

```
电影感治愈风格，9:16竖屏，冷蓝回忆色调
@图片1首帧，@图片2角色参考

【镜头】中景全身 → 缓缓抬头 → 转身离去
【主体】回忆中女孩撑透明伞在便利店门口
【变化】低头等待 → 抬头望街道 → 失望转身
【光影】冷白霓虹，雨夜蓝调，孤独光晕
【声音】忧伤大提琴 + 雨声渐强 + "我等的人没有来"
```

### 尾帧描述
小晚撑着透明伞转身离开，背影消失在雨幕中。便利店霓虹在雨水中晕染成模糊光斑。

### 导演备注
- **情绪基调**: 怀旧、淡淡的忧伤、孤独
- **关键动作**: 从等待的静止到转身离开，体现希望落空
- **色调提示**: 冷蓝色调，与E01暖调形成对比，暗示回忆
- **衔接提示**: 背影转身动作可自然过渡到E03回到直播间
