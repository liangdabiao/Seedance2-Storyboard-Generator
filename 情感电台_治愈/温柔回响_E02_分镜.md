### 上传素材

| 槽位 | 素材编号 | 内容 |

| @图片1（首帧） | **KF03** | 雨中回忆 — 透明伞下侧影，便利店霓虹 |
| @图片2（参考） | **C03** | 小晚·回忆雨天造型 — 角色造型参考 |
| @图片3（参考） | **S02** | 雨天街道·便利店外 — 场景氛围参考 |

### Seedance Prompt

```
电影感治愈风格，9:16竖屏，冷蓝回忆色调
@图片1首帧，@图片2角色参考，@图片3场景参考

【镜头】中景全身 → 缓缓抬头 → 转身离去
【主体】回忆中小晚撑透明伞等便利店门口
【变化】低头等待 → 抬头望向街道 → 失望转身
【光影】便利店冷白霓虹，雨夜蓝调，孤独光晕
【声音】忧伤大提琴 + 雨声渐强 + 汽车驶过水花声 + "那年的雨下得很大，我等的人没有来..."
```

### 尾帧描述
小晚撑着透明伞转身离开，背影消失在雨幕中。便利店霓虹在雨水中晕染成模糊的光斑，街道空无一人。

### 导演备注
- **情绪基调**: 怀旧、淡淡的忧伤、孤独
- **关键动作**: 从等待的静止到转身离开的动态，体现希望落空
- **色调提示**: 整体偏冷蓝色调，与E01的暖调形成对比，暗示这是回忆
- **衔接提示**: 尾帧的背影转身动作可自然过渡到E03回到直播间读信的场景
