### Seedance Prompt

```
电影感治愈风格，9:16竖屏
@图片1首帧，@图片2角色参考

【镜头】侧脸特写 → 微转头望向窗外
【主体】主播小晚深夜独自开播
【变化】凝视前方 → 眼神柔软 → 望向窗外雨滴
【光影】暖色台灯侧光，窗外冷色霓虹虚焦
【声音】安静钢琴 + 雨声 + 键盘声 + "这里是深夜电台温柔回响"
```

### 尾帧描述
小晚望向窗外雨滴，眼神温柔而遥远。窗外城市灯火模糊成光斑，她陷入回忆。

### 导演备注
- **情绪基调**: 静谧、略带孤独、温柔
- **关键动作**: 从望向镜头方向自然转向窗外，营造故事即将开始的感觉
- **衔接提示**: 尾帧的雨滴和窗外景色将自然过渡到E02的雨夜回忆场景
