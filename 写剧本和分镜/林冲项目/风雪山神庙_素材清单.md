# 风雪山神庙 - 素材清单

## 视觉风格定义

**统一风格前缀：**
```
2D中国动漫风格，类似《大护法》《雾山五行》，赛璐珞阴影处理，线条清晰，色彩鲜明但沉稳，扁平化配色，电影感构图，富有表现力的角色设计，动感和张力十足
```

## 六、风格一致性说明

1. **色彩统一：** 所有素材采用中国传统水墨画的黑白灰主色调，适当使用色彩点缀（如枪缨的红色、火焰的橙红色）

2. **光影处理：** 戏剧化的光影对比，明暗分明，突出重点

3. **质感表现：** 雪的质感、皮的质感、金属的质感都要清晰可辨

4. **氛围营造：** 风雪的苍凉感、英雄的孤独感、爆发的激烈感要贯穿始终

5. **角色识别：** 林冲的绿色战袍、毡笠、花枪是识别标志；陆谦的阴鸷表情、深色官服是识别标志
