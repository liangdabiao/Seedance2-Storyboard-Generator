# 风雪山神庙项目 - 使用指南

## 项目概述

本项目将《水浒传》中林冲的经典故事"风雪山神庙"改编为5集AI短视频系列，展现林冲从隐忍到反抗的彻底转变。

## 制作流程

### 第一步：资产生成

使用 **Nana Banana Pro** 按照素材清单生成所有需要的图像资产。

**资产生成要点：**
1. 统一使用风格前缀："中国传统水墨画风格结合写实渲染，黑白灰为主色调"
2. 角色素材需要多角度（C01-C11）
3. 场景素材需要日夜景区分（S01-S09）
4. 道具素材注意细节表现（P01-P10）
5. 每个资产编号对应具体用途，不可混淆

### 第二步：视频生成

使用 **Seedance 2.0** 平台生成5集视频。

**第1集生成（无需视频延长）：**
1. 上传对应图片到图片1-图片6位置
2. 复制 E01 分镜中的 Seedance Prompt
3. 生成15秒视频

**第2-5集生成（使用视频延长功能）：**
1. 上传上一集视频到@视频1位置
2. 上传对应图片到图片位置
3. 修改 Prompt 开头为："将@视频1延长15s，"
4. 保持后续描述不变
5. 生成15秒视频，实现无缝衔接

### 第三步：后期处理

1. **字幕添加：** 根据【字幕】信息添加中文字幕
2. **声音调整：** 确保音乐、音效、对话的平衡
3. **集间衔接：** 检查各集之间的连贯性
4. **整体调色：** 统一全剧的黑白灰水墨风格

## 关键技术要点

### 风格一致性

- **色调：** 赛璐珞风格，清晰轮廓线，扁平化配色，红色仅用于点缀（血、火、枪缨）
- **光影：** 明确的阴影块面，类似动画的赛璐珞阴影处理
- **线条：** 清晰有力的轮廓线，角色和场景线条一致

### 情感递进

| 集数 | 核心情感 | 视觉表现 |
|------|----------|----------|
| 1 | 孤寂、压抑 | 大面积留白，人物渺小 |
| 2 | 危机、困顿 | 动态镜头，混乱场景 |
| 3 | 震惊、觉醒 | 眼神特写，内外光线对比 |
| 4 | 爆发、宣泄 | 快速剪辑，红白对比 |
| 5 | 坚定、新生 | 晨光出现，步伐坚定 |

### Seedance 2.0 约束

- **最多9张图片**：每集分镜已控制在9张以内
- **最多3个参考视频**：第2-5集仅需1个参考视频（上一集）
- **避免敏感词**："官府"、"朝廷"等词可适当替换
- **指令长度**：每集Prompt控制在合理长度

## 常见问题解决

### 问题1：图片生成风格不一致

**解决：** 确保每个提示词都包含完整的风格前缀："2D中国动漫风格，赛璐珞阴影处理，线条清晰，扁平化配色"

### 问题2：视频衔接不流畅

**解决：** 第2-5集必须使用视频延长功能，且尾帧描述要准确，确保下一集能从尾帧状态开始

### 问题3：风雪效果不明显

**解决：** 在Prompt中强调"大雪纷飞"、"漫天飞雪"、"风雪交加"等关键词

### 问题4：打斗场面不够激烈

**解决：** 使用动态镜头语言，强调"快速剪辑"、"慢镜头特写"、"动作残影"等技巧
